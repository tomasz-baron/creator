#Creator Application

### Node.js and Tools

- Get [Node.js][node].
- Install the tool dependencies: `npm install`

### Running the Application
- Run `npm run serve`
- Address: http://localhost:8000

### Unit Testing
- Start tests with `npm run test`.

### Tested on
- Node.js v8.12.0
- Chromium 70.0.3538.77
- Ubuntu 18.04.1 LTS
